## Description
JobJunction is a MERN Stack based job portal website where users can browse jobs, post job listings and apply for jobs.  
The platform supports secure authentication using email/password and Google login via Firebase.

It is designed to connect job seekers and employers on a single platform with an easy-to-use interface.

---

## Features

### User
- User Signup & Login
- Google Authentication (Firebase)
- Browse job listings
- Filter jobs by title, location & time
- Post and manage job listings
- Pagination support

### Admin
- Admin login
- Manage job postings
- Manage users
- Content moderation
